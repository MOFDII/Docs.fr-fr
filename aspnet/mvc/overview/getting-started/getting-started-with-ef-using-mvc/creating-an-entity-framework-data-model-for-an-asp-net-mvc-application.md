## <a name="conventions"></a><span data-ttu-id="291e1-329">Conventions</span><span class="sxs-lookup"><span data-stu-id="291e1-329">Conventions</span></span>

<span data-ttu-id="291e1-330">La quantité de code que vous deviez écrire dans l’ordre pour Entity Framework pouvoir créer une base de données complète pour vous est minimale en raison de *conventions*, ou hypothèses qui rend d’Entity Framework.</span><span class="sxs-lookup"><span data-stu-id="291e1-330">The amount of code you had to write in order for Entity Framework to be able to create a complete database for you is minimal because of *conventions*, or assumptions that Entity Framework makes.</span></span> <span data-ttu-id="291e1-331">Certains d'entre eux ont déjà été observées ou ont été utilisées sans votre connaître les :</span><span class="sxs-lookup"><span data-stu-id="291e1-331">Some of them have already been noted or were used without your being aware of them:</span></span>

- <span data-ttu-id="291e1-332">Les formulaires pluralisées des noms de classes d’entité sont utilisés comme noms de tables.</span><span class="sxs-lookup"><span data-stu-id="291e1-332">The pluralized forms of entity class names are used as table names.</span></span>
- <span data-ttu-id="291e1-333">Les noms de propriété d’entité sont utilisées pour les noms de colonne.</span><span class="sxs-lookup"><span data-stu-id="291e1-333">Entity property names are used for column names.</span></span>
- <span data-ttu-id="291e1-334">Propriétés de l’entité qui sont nommées `ID` ou *classname* `ID` sont reconnus comme des propriétés de clé primaire.</span><span class="sxs-lookup"><span data-stu-id="291e1-334">Entity properties that are named `ID` or *classname* `ID` are recognized as primary key properties.</span></span>
- <span data-ttu-id="291e1-335">Une propriété est interprétée comme une propriété de clé étrangère si elle se nomme *&lt;nom de propriété de navigation&gt;&lt;nom de propriété de clé primaire&gt;* (par exemple, `StudentID` pour la `Student` propriété de navigation depuis le `Student` clé primaire de l’entité est `ID`).</span><span class="sxs-lookup"><span data-stu-id="291e1-335">A property is interpreted as a foreign key property if it's named *&lt;navigation property name&gt;&lt;primary key property name&gt;* (for example, `StudentID` for the `Student` navigation property since the `Student` entity's primary key is `ID`).</span></span> <span data-ttu-id="291e1-336">Propriétés de clé étrangère peuvent également être nommées les mêmes simplement &lt;nom de propriété de clé primaire&gt; (par exemple, `EnrollmentID` dans la mesure où le `Enrollment` clé primaire de l’entité est `EnrollmentID`).</span><span class="sxs-lookup"><span data-stu-id="291e1-336">Foreign key properties can also be named the same simply &lt;primary key property name&gt; (for example, `EnrollmentID` since the `Enrollment` entity's primary key is `EnrollmentID`).</span></span>
