<a name="how-do-i-use-the-aspnet-ajax-timer-control"></a>[Comment faire] Utiliser le contrôle Timer d’ASP.NET AJAX ?
====================
par [Joe Stagner](https://github.com/JoeStagner)

Le contrôle Timer d’AJAX permet une partie d’une page web ASP.NET à mettre à jour dynamiquement à intervalles réguliers, au lieu d’utiliser l’utilisateur d’effectuer une action telles qu’un clic sur un bouton.
