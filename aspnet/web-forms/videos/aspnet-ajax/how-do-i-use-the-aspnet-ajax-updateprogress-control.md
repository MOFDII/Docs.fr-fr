<a name="how-do-i-use-the-aspnet-ajax-updateprogress-control"></a>[Comment faire] Utiliser le contrôle UpdateProgress d’ASP.NET AJAX ?
====================
par [Joe Stagner](https://github.com/JoeStagner)

Dans une application AJAX, il est souvent un temps de pause entre l’heure que le navigateur envoie une demande réseau et le temps que le navigateur reçoit une réponse. Pour informer l’utilisateur que la conversation réseau a lieu, le UpdateProgress AJAX ASP.NET contrôle fournit des commentaires immédiats en affichant une image animée, texte « chargement » ou autres HTML qui disparaît une fois que la réponse du réseau.
