## <a name="avoiding-cross-site-scripting-xss-attacks"></a><span data-ttu-id="0f04a-152">En évitant les attaques par écriture de scripts entre sites (XSS)</span><span class="sxs-lookup"><span data-stu-id="0f04a-152">Avoiding Cross-Site Scripting (XSS) Attacks</span></span>
