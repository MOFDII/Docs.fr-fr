### <a name="partial-view-discovery"></a><span data-ttu-id="fc6bf-134">Découverte des vues partielles</span><span class="sxs-lookup"><span data-stu-id="fc6bf-134">Partial View Discovery</span></span>

<span data-ttu-id="fc6bf-135">Quand vous référencez une vue partielle, vous pouvez faire référence à son emplacement de plusieurs façons :</span><span class="sxs-lookup"><span data-stu-id="fc6bf-135">When referencing a partial view, you can refer to its location in several ways:</span></span>

```text
// Uses a view in current folder with this name
// If none is found, searches the Shared folder
@Html.Partial("ViewName")

// A view with this name must be in the same folder
@Html.Partial("ViewName.cshtml")

@Html.Partial("../Account/LoginPartial.cshtml")
```
