# <a name="adding-a-controller-to-an-aspnet-core-mvc-app-with-visual-studio-code"></a>Ajout d’un contrôleur à une application ASP.NET Core MVC avec Visual Studio Code

Par [Rick Anderson](https://twitter.com/RickAndMSFT)

[!INCLUDE[adding-controller](../../includes/mvc-intro/adding-controller1.md)]

* Dans **VS Code**, sélectionnez l’icône **EXPLORATEUR** et cliquez en maintenant enfoncée la touche Ctrl (ou cliquez avec le bouton droit) sur **Contrôleurs > Nouveau fichier** et nommez le nouveau fichier HelloWorldController.cs.
