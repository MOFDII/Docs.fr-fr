> [!NOTE]
> Si la commande `dotnet new webapp <OPTIONS>` charge l’aide de la commande [dotnet new](/dotnet/core/tools/dotnet-new) au lieu de créer une application Razor Pages, installez le [SDK .NET Core 2.1.300 ou ultérieur](https://www.microsoft.com/net/download/archives). À compter du SDK .NET Core 2.1.300, le *nom court* `webapp` est un alias pour `razor`.
